# Rio Airbnb Pricing Lab

Pricing decision tool for Airbnb hosts in Rio de Janeiro, built on open Inside Airbnb data.
Insight-first README and live dashboard land in Phase 3. See `docs/superpowers/`.

**Data snapshot:** `2026-03-30` (Inside Airbnb Rio de Janeiro). Resolved at runtime and
recorded — with per-file `sha256` — in `data/curated/manifest.json`. The Phase-1 pipeline
produces 39,816 analyzable listings + a listing × month × day-of-week seasonality table.
